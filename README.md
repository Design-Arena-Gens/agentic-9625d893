# NeuraForge Hero

High-impact hero experience for an enterprise workflow intelligence platform built with Next.js 14 and Tailwind CSS.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run start
```

## Deploy

This project is optimised for Vercel. To deploy a production build run:

```bash
vercel deploy --prod --yes --token $VERCEL_TOKEN --name agentic-9625d893
```
